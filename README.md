### top-deals

- A Sinatra App for parsing hotel deals

- ruby v2.2.5

- Sinatra v2.0.0

- [Access app on Heroku](https://evening-cliffs-85685.herokuapp.com/)

### Installation Instructions

```sh
$ sudo dnf install ruby rubygems
$ gem install bundle
$ cd top-deals
$ bundle install
$ ruby app.rb
```
